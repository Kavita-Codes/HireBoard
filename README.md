# HireBoard — Candidate Management Dashboard

HireBoard is a lightweight hiring dashboard built using **HTML, CSS, and Vanilla JavaScript**.  
It allows users to manage candidate workflows with dynamic UI updates and persistent storage without using any frontend frameworks.

This project demonstrates real-world frontend fundamentals such as DOM manipulation, modular architecture, and state-like data handling.

## Features:

- Add candidates with role and status
- Update hiring stage (Applied → Interview → Selected → Rejected)
- Delete candidates dynamically
- Live dashboard statistics
- Scrollable candidate list for large datasets
- Data persistence using LocalStorage
- Modular JavaScript structure (ES Modules)
- Responsive and modern UI design
- No frameworks — built using pure JavaScript

## Tech Stack

- HTML5  
- CSS3 (Modern Layout + UI Styling)  
- Vanilla JavaScript (ES6 Modules)  
- LocalStorage API (Data Persistence)
